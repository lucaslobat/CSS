padding

The padding CSS shorthand property sets the padding area on all four sides of an element at once. An element's padding area is the space between its content and its border.
Note: Padding creates extra space within an element. In contrast, margin creates extra space around an element.
Constituent properties This property is a shorthand for the following CSS properties:
  • padding-bottom
  • padding-left
  • padding-right
  • padding-top
